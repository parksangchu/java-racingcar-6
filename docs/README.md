## 기능 구현 목록
***
- [ ] 자동차 이름을 입력 받는다.
- [ ] 자동차 이름을 쉼표로 구분한다. 각자 이름은 5자 이하만 가능하다.
- [ ] 자동차가 몇 번 이동할지 입력 받는다.
- [ ] 입력 받은 이동 횟수만큼 해당 행위를 반복한다.
  - [ ] 각 자동차마다 0~9 사이의 무작위 값을 발생시킨다.
  - [ ] 값이 4이상이면 자동차를 전진시킨다.
  - [ ] 자동차의 이름과 결과를 출력한다.
- [ ] 우승자를 확인한다.
- [ ] 우승자를 출력한다.
  - [ ] 우승자가 여려 명일 경우 쉼표를 이용하여 구분한다. 
- [ ] 잘못된 값을 입력받으면 예외를 발생시키고 프로그램을 종료한다.
